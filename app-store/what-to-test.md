# What to Test — Memento v1.0.0

## Core Features

### Life Screen (Tab 1)
- View life progress percentage and weeks remaining
- Life grid visualization with milestone markers
- First launch: birthday onboarding flow (enter YYYY/MM/DD)

### Wisdom Screen (Tab 2)
- Daily Stoic quote display
- Tap heart to favorite/unfavorite quotes (gold = favorited)
- Tap arrow to see next quote
- Tap share to share quote via system share sheet

### Settings Screen (Tab 3)
- Edit birthday (tap date row → expand inline editor)
- Adjust life expectancy slider (60-100 years)
- View milestones (Graduated, Married, First Home)
- Add milestones from template list
- Toggle Daily Quote notification
- Upgrade to Pro / view Pro status

### Paywall
- Tap "Upgrade to Pro" in Settings
- Verify pricing displays correctly
- Test purchase flow (Sandbox)
- Close button (X) returns to Settings

## Known Limitations
